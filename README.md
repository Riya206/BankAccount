**Bank Account**

**Project Overview:**

The Bank Account Management System is a Java-based application developed using Core Java and layered architecture principles. It allows users to perform basic banking operations such as checking account balance and transferring money between accounts. The system uses Bean classes to store account and transaction data and Service classes to handle business logic. The project follows a layered architecture pattern for better maintainability and clarity.

**Technologies Used**
Java (JDK 8 or above)
Core Java
Collections 
Framework 
OOPS Concepts 
Git & GitHub

**Architecture:**
Used Bean Layer 
Data representation Service Layer 
Business logic Main Layer 
Application execution
